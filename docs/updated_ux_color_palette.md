# Suggested UX Usage

- Darkest shades → icons, active states, highlights
- Middle shades → cards, illustrations, secondary buttons
- Lightest shades → backgrounds, tags, hover states

This palette has a Scandinavian / soft enterprise SaaS feel — calm, trustworthy, and modern.
